# FitKit - 智能健身工具箱

**无需联网、保护隐私**的私人健身伴侣。将所有训练中需要的零散功能整合于一体，让手机变成随身的智能教练。

## 功能模块

### 🎬 视频动作分析
基于端侧 AI 引擎（MoveNet）的实时姿态估计，**完全离线可用**：
- 支持摄像头实时采集、示例视频、上传文件、视频 URL 四种输入
- 实时动作计数（次数）与运动阶段识别
- 动作标准度反馈（良好 / 注意 / 偏差）
- 分析结束后自动生成改进建议
- 内置深蹲、哑铃弯举、肩上推举三种动作模式

### ⏱️ 间隔计时器
可自定义训练/休息时长的组间计时器，适合 HIIT、Tabata 等间歇训练。

### 📊 训练记录
自动保存每次训练的日期、动作、次数数据，方便追踪训练进展。

### 📖 动作库
标准动作要领与常见错误对照，可展开查看每个动作的详细说明和注意事项。

## 技术栈

| 层面 | 技术 |
|------|------|
| UI | 纯 HTML/CSS/JS，深色主题 + 移动端响应式 |
| 姿态估计 | TensorFlow.js + MoveNet (Thunder) |
| 视频处理 | Canvas API 实时渲染骨骼关键点 |
| 部署 | 纯静态文件，CDN 即可托管 |

## 快速开始

```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

无需构建工具，无需安装依赖。首次使用视频分析功能时会自动下载 MoveNet 模型（约 10MB），后续使用浏览器缓存加载。

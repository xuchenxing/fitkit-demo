# FitKit - 智能健身工具箱

**无需联网、保护隐私**的私人健身伴侣。将所有训练中需要的零散功能（计时、记录、分析、计划）整合于一体，让手机变成随身的智能教练。

## 特性

- 完全离线可用，无需注册登录
- 视频动作分析，实时动作标准度评估
- 组间休息计时器、训练记录
- 所有数据在设备端计算，杜绝隐私焦虑
- 深色主题 UI，移动端适配

## AI 能力

本项目端侧 AI 由 Google MoveNet (Thunder) 姿态估计模型驱动，所有推理在浏览器内完成：

- **实时骨骼关键点检测** — 17 个身体关键点，30+ FPS
- **动作阶段识别** — 自动判断上举/下放阶段，精准计数
- **关节角度计算** — 基于关键点间向量，评估动作幅度与对称性
- **标准度反馈** — 规则引擎 + 阈值判定，实时提示动作质量

## 使用方式

直接用浏览器打开 `index.html` 即可运行，无需构建、无需安装依赖。

首次使用视频分析时需下载 MoveNet 模型（约 10MB），后续使用缓存。

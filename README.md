# FitKit - 智能健身工具箱

**无需联网、保护隐私**的私人健身伴侣。将所有训练中需要的零散功能（计时、记录、分析、计划）整合于一体，让手机变成随身的智能教练。

## 特性

- 完全离线可用，无需注册登录
- 视频动作分析，实时动作标准度评估
- 组间休息计时器、训练记录
- 所有数据在设备端计算，杜绝隐私焦虑
- 深色主题 UI，移动端适配

## 项目结构

## 使用方式

直接用浏览器打开 `index.html` 即可运行，无需构建、无需安装依赖。

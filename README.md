# FitKit - 智能健身工具箱

**无需联网、保护隐私**的私人健身伴侣。将所有训练中需要的零散功能整合于一体，让手机变成随身的智能教练。

## 功能模块

### 🎬 视频动作分析
基于端侧 AI 引擎（MoveNet）的实时姿态估计，**完全离线可用**：
- 支持摄像头实时采集、示例视频、上传文件、视频 URL 四种输入
- 实时动作计数（次数）与运动阶段识别
- 动作标准度反馈（良好 / 注意 / 偏差）
- 分析结束后自动生成改进建议
- 内置深蹲、哑铃弯举、肩上推举三种动作模式

### ⏱️ 间隔计时器
可自定义训练/休息时长的组间计时器，适合 HIIT、Tabata 等间歇训练。

### 📊 训练记录
自动保存每次训练的日期、动作、次数数据，方便追踪训练进展。

### 📖 动作库
标准动作要领与常见错误对照，可展开查看每个动作的详细说明和注意事项。

## 技术栈

| 层面 | 技术 |
|------|------|
| UI | 纯 HTML/CSS/JS，深色主题 + 移动端响应式 |
| 姿态估计 | TensorFlow.js + MoveNet (Thunder) |
| 视频处理 | Canvas API 实时渲染骨骼关键点 |
| 部署 | 纯静态文件，CDN 即可托管 |

## AI 能力

本项目端侧 AI 由 Google MoveNet (Thunder) 姿态估计模型驱动，所有推理在浏览器内完成：

- **实时骨骼关键点检测** — 17 个身体关键点，30+ FPS
- **动作阶段识别** — 自动判断上举/下放阶段，精准计数
- **关节角度计算** — 基于关键点间向量，评估动作幅度与对称性
- **标准度反馈** — 规则引擎 + 阈值判定，实时提示动作质量

## 支持的动作

| 动作 | 英文名 | 目标肌群 | 关键角度 |
|------|--------|----------|----------|
| 深蹲 | Squat | 股四头肌、臀部 | 膝盖弯曲角度、躯干前倾角 |
| 哑铃弯举 | Bicep Curl | 肱二头肌 | 肘关节角度、手腕偏移 |
| 肩上推举 | Shoulder Press | 三角肌 | 肩关节外展角、肘部伸展 |

## 离线隐私

- 摄像头采集仅在本地浏览器内处理，**视频不会上传到任何服务器**
- MoveNet 模型首次加载后由浏览器 IndexedDB 缓存，此后完全离线
- 训练记录存储在 localStorage 中，清除浏览器数据即删除

## 已知限制

- 多人场景下姿态检测可能不稳定，建议单人使用
- 暗光环境会影响关键点识别精度
- iOS Safari 摄像头模式需用户手动授权

## 快速开始

```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

无需构建工具，无需安装依赖。首次使用视频分析功能时会自动下载 MoveNet 模型（约 10MB），后续使用浏览器缓存加载。
